Castlevania Remake

As part of an exam in 2016, I remade the first level of the classic 1987 release, Castlevania from scratch. My goal in this project was to learn about some of the features of C++ I had not yet used before such as Inheritance and Polymorphism. Additionally, I learned about XML in implementing a system to populate the level with enemies and pickups by reading a file.

This project was made in C++ using a custom framework provided by school.
